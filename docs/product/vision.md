# Visão do Produto

`private-ai-companion` deve ser uma base open-source para criar uma presença virtual privada no desktop.

A visão é permitir que qualquer pessoa configure sua própria companion AI, com personalidade, voz, avatar, memória local e interação segura, sem depender de live, público ou plataformas de streaming.

## Experiência desejada

A companion deve parecer uma presença viva, mas respeitosa:

- responde quando chamada;
- pode ser proativa dentro de limites configuráveis;
- lembra preferências de forma controlada;
- expressa emoções visualmente;
- fala com naturalidade;
- não invade privacidade;
- não executa ações perigosas;
- pode ser personalizada.
