# Não Objetivos

Este projeto deliberadamente não pretende:

- competir com plataformas de streaming;
- ser bot de audiência;
- automatizar todo o sistema operacional;
- ser assistente corporativo genérico;
- usar uma personalidade fixa;
- exigir hardware específico;
- exigir provedor cloud específico;
- salvar tudo que o usuário diz;
- esconder envio de dados para APIs externas.
