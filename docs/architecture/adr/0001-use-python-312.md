# ADR-0001 — Use Python 3.12+

## Status

Accepted

## Contexto

Python tem o ecossistema mais adequado para LLMs, STT, TTS, automação desktop, SQLite e tooling de IA.

## Decisão

O projeto usará Python 3.12+ como linguagem principal.

## Consequências

Ganha-se produtividade e integração com IA/voz; exige disciplina de tipagem e testes.
