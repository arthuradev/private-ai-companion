# ADR-0014 — Privacy-first public project

## Status

Accepted

## Contexto

O projeto será público e usado por pessoas diferentes.

## Decisão

Não haverá dados pessoais, caminhos fixos ou personalidade obrigatória; privacidade será padrão.

## Consequências

Aumenta adoção e reduz risco de vazamento/configuração insegura.
